Technical Blog: Comparing Arrays and Hashes in Ruby

March 15, 2015

An array is a numbered collection of objects. They are useful when storing numbers, strings, other arrays, hashes, and classes. To store values, you can simply use square brackets, []. I find it useful because it allows you to select an object based on a consistent, consecutive numerical index. There are multiple ways to create arrays, such as Array.new, .to_a, or by using a simple [].

Arrays have their own methods to manipulate their values. For example: `.push, .pop, .first, .last, .at` etc. To access a value in an array, you can state the number index where its ordered, BUT the numbers must start from zero. Here is an example of an array: array = [1, 2, 3, 4, 5] #if I wanted to find the value of 1, I would use, array.at(0) #and this would return 1, => 1

In addition, you can have arrays within an array. For example; array_1 = [1, 2, 3, 4, 5, [2, 6, 7, 3, 5]] array_2 = [[1, 2, 3, 4, 5], [2, 6, 7, 3, 5]] Also, you can merge them into one array by using the array method of .flatten. For example; array_1 = [1, 2, 3, 4, 5, [2, 6, 7, 3, 5]].flatten => [1, 2, 3, 4, 5, 2, 6, 7, 3, 5] You can find out how many values an array has stored in it by using .length. For example; array = [1, 2, 3, 4, 5].length => 5 You can pull an unique value by using .uniq or you can join them together by using join. array = [1, 1, 2, 3, 3, 4, 4, 4, 5, 5].uniq => 2 array = [1, 2, 3, 4, 5].join => 12345

Hash is an unordered collection of objects that pairs objects by a key and a value. It's similar to an array, but instead of numbered indexes, a hash stores values by unique keys. For example: {key: value} Hashes are useful when storing objects like numbers, strings, arrays, classes, etc. To store a value you need to use curly brackets {}. You can also select an object based on the given key it was stored under. For example: fruit_colors = {apple: red, banana: yellow, kiwi: green} print fruit_colors(apple) => red

You can create a hash with Hash.new, #to_hash or with simple {}. Hashes have their own methods to manipulate their values (like, .delete, .freeze, .fetch, etc.). You can also merge multiple hashes into one hash by using hash method .merge. For example; x1 = { "a" => 100, "b" => 200 } x2 = { "b" => 254, "c" => 300 } x1.merge(x2) returns {"a"=>100, "b"=>254, c"=>300}

You can find out how many values a given hash is storing by stating .length. For example: x1 = { "a" => 100, "b" => 200 }.length => 2
